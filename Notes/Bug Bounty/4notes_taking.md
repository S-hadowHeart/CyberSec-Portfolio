### During Hacking Notes
- __Goal:__ To record things like IDs or endpoints i'll need later
- __FOrmat:__ Text and requests
- __Requirements:__
  - Be able to link high level functionallity to requests
  - Keep track of multiple IDs for testing
  - TO be able to come back to it later
  - Doesn't have to be digital

### Knowledge Base Notes
- __Goal:__ A searchable note system that i can use while hacking
- __Format:__ Links, bullet points, notes
- __Requirements:__
  - Digital for searching
  - Flexible so i can attach payloads URLs, notes
  - Allow me to store my previous finding AND notes
  - Be able to have some organisation, e.g by vulnerability class
