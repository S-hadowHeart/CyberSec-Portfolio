## Subnet Mask
- it is a 32 bit number used to indicate number to indicated number of bits borrow from host-id and there positions based on the following

- __Rule1:__ Number of 1's in the subnet maskk indicate `NID + SID`
- __Rule2:__ Number of 0's in the subnet maskk indicate HID part

## Default subnet mask : 
- Class A : 255.0.0.0
- Class B :  255.255.0.0
- Class C : 255.255.255.0
